# Graphyne Brand Assets

Replace these files to customize the app brand:

- `favicon-light.svg` - browser favicon used in light mode
- `favicon-dark.svg` - browser favicon used in dark mode
- `logo-mark-light.svg` - square mark used in light mode UI
- `logo-mark-dark.svg` - square mark used in dark mode UI
- `logo-light.svg` - full horizontal logo for light mode
- `logo-dark.svg` - full horizontal logo for dark mode
- `favicon.svg` - metadata fallback that also respects system color scheme

Keep the filenames stable unless you also update `app/layout.tsx`.
